🧩 Next.js Figma UI Implementation Task

This project is a frontend implementation built as part of a UI/technical assessment. The goal was to convert a Figma design into a fully functional and responsive web page using Next.js and modern frontend practices.

This project demonstrates my ability to work with design specifications, build reusable components, and implement interactive UI features.

🚀 Tech Stack
Next.js 14+ (App Router)
TypeScript
Tailwind CSS
Swiper.js
React Hook Form
Zod Validation
React Hot Toast
📌 Project Features
1. Hero Section

A simple hero section based on the provided Figma design, including image and text layout with responsive styling.

2. Swiper Carousel (Common Obstacles Section)

Implemented using Swiper.js with the following features:

Autoplay enabled
Navigation arrows (next / previous)
Pause on hover functionality
Smooth and responsive sliding experience
3. Consultation Form

A fully functional form section with:

Controlled inputs using onChange
Local state management
Async submit handler (2-second simulated request)
Form validation:
Required field validation
Error messages displayed under inputs
Prevents submission if validation fails
4. FAQ Section (Accordion)

An interactive accordion component featuring:

Expand/collapse functionality
Toggle open/close using "+" and "-" buttons
Clean UX behavior for multiple questions
🎯 Project Requirements (Followed)
Next.js 14+ with App Router
Tailwind CSS for styling only
Swiper.js for carousel functionality
Clean and modular component structure
Responsive design implementation
